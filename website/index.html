Structural Inspection Drone

An autonomous approach to building inspection.

The Idea

The goal of this project, in collaboration with Northern Arizona University's Dr. Abolfazl Razi, is to design an autonomous drone which is capable of inspecting designated structures and identifying areas of concern. This drone should be able to fly around the building without direct control, and should use collision avoidance and safe landing features to avoid damage to the structure and drone. The drone will additionally be operable using a simple Android app, which will allow the user to set flight paths and observe video footage.
